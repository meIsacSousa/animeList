# animeList
###   Projeto desenvolvido para a disciplina de Banco de Dados.

   Projeto consiste em um sistema para listagem e organização de conteúdos que o usuário está acompanhando, já assistiu ou que possui interesse em assistir, projeto possui alguns conceitos de banco de dados como gatilhos, regras e funções, para funcionamento adequado da aplicação é necessário configurar banco de dados de acordo.
